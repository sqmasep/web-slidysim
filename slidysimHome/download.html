How to install?

If you are downloading the program for the first time, or updating from a pre-v28.2 version to the latest version, follow these steps

  1. Download Libraries and Binary
  2. Create a new 'slidysim' folder
  3. Extract the contents of Libraries into the 'slidysim' folder
  4. Extract the program executable from Binary into the 'slidysim' folder
  5. If you are updating from an old version to v28.2, copy the following files and folders (if you have them) from your old (pre-v28.2) slidysim folder into your new one: images (folder), save (folder) schemes (folder), account.txt, pbs.csv, settings.txt. Delete the old slidysim folder
  6. Run the program
